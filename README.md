# 🚀 JJBench

一款简洁、中文友好的 VPS 一键体检脚本  
A simple Chinese VPS benchmark & diagnostic tool

---

## ✨ 项目简介

**JJBench** 是一个轻量级 VPS 体检工具，专为中文用户设计。

设计目标：

- 简洁
- 直观
- 无复杂依赖
- 一条命令即可运行

适用于：

- 🆕 新购 VPS 体检
- 🔍 服务器巡检
- ⚙️ 快速环境确认
- 🧪 轻量性能测试
- ☁ 云服务器基础状态排查

---

## 📦 当前功能 (v0.2)

### 📦 系统信息

- 主机名
- 操作系统
- 内核版本
- 系统架构
- 运行时间（中文显示）
- 虚拟化类型识别

---

### 🧠 CPU 信息

- CPU 型号
- 核心数量
- 主频

---

### 💾 内存信息

- 总内存
- 使用情况
- Swap 状态检测

---

### 🗄 磁盘信息

- 磁盘大小
- 文件系统类型
- 使用率

---

### 🌐 网络信息

- 内网 IPv4
- 公网 IPv4
- 公网 IPv6
- NAT / 直连识别
- TCP 拥塞算法检测（BBR / cubic）

---

### 🚀 I/O 磁盘写入测试

- 真实写入测试（绕过缓存）

---

## 🚀 一键运行

无需下载文件，直接执行：

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/EnockLee/JJBench/main/jjbench.sh)
```
